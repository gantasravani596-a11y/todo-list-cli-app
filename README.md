# To-Do List CLI Application

## 📌 Objective
This project is a simple console-based To-Do List Manager built using Python.

## 🛠 Tools Used
- Python
- VS Code / Terminal

## ✨ Features
- Add tasks
- View tasks
- Remove tasks
- Tasks stored in file (persistent storage)

## 📂 Files
- todo.py → main program
- tasks.txt → stores tasks

## ▶️ How to Run
```bash
python todo.py
